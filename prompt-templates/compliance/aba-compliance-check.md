**模板版本**: v1.0  
**适用范围**: ABA职业道德规范合规检查  
**更新日期**: 2025-08-11

## ABA职业道德规范核心要求

⚠️ **强制性合规声明**: 本模板用于确保所有AI辅助法律服务严格符合ABA Model Rules of Professional Conduct的要求。每个AI工具的使用和输出都必须经过本合规检查，确保维护律师职业的最高道德标准。

## Rule 1.1 - 胜任能力 (Competence)
### 技能和知识要求
- **法律知识**: AI辅助必须以执业律师的法律知识为基础
- **技能要求**: 律师必须具备处理相关法律事务的技能
- **准备充分**: 在提供法律服务前必须进行充分准备
- **持续学习**: 跟上法律发展和技术进步

### AI工具使用的胜任能力要求
- **工具理解**: 律师必须充分理解AI工具的能力和局限性
- **输出验证**: 必须验证和确认AI生成的所有法律分析
- **专业判断**: AI输出不能替代律师的专业判断
- **质量控制**: 建立有效的质量控制机制

### 合规检查要点
- [ ] 是否明确标注需要律师验证
- [ ] 是否避免超出律师能力范围的建议
- [ ] 是否提供了充分的法律依据
- [ ] 是否包含了适当的不确定性说明

## Rule 1.6 - 信息保密 (Confidentiality)
### 保密义务范围
- **客户信息**: 所有从客户获得的信息
- **代表过程**: 律师代表客户过程中获得的信息
- **推断信息**: 基于客户信息进行的推断和分析
- **第三方信息**: 与代表客户相关的第三方信息

### AI工具使用的保密要求
- **数据输入**: 避免在AI提示中包含客户敏感信息
- **数据存储**: 确保AI服务提供商的数据安全
- **数据传输**: 保证数据传输的安全性
- **数据删除**: 确保数据的及时删除和销毁

### 合规检查要点
- [ ] 是否明确提醒避免输入客户敏感信息
- [ ] 是否有数据脱敏处理指导
- [ ] 是否明确数据保护责任
- [ ] 是否包含保密义务提醒

## Rule 1.7 - 利益冲突: 当前客户 (Current Client Conflicts)
### 利益冲突识别
- **直接对立**: 代表的客户利益直接对立
- **重大限制**: 对一个客户的代表受到另一客户利益的重大限制
- **个人利益**: 律师个人利益与客户利益冲突
- **第三方利益**: 第三方利益对律师判断的影响

### AI工具相关的利益冲突
- **多客户服务**: 同时为多个客户提供AI辅助服务
- **信息交叉**: 不同客户信息的交叉污染
- **算法偏见**: AI算法可能存在的偏见
- **商业利益**: AI服务提供商的商业利益

### 合规检查要点
- [ ] 是否有利益冲突检查程序
- [ ] 是否避免了信息交叉风险
- [ ] 是否明确了律师独立判断要求
- [ ] 是否包含利益冲突披露要求

## Rule 1.9 - 利益冲突: 前客户 (Former Client Conflicts)
### 前客户保护
- **相同事项**: 不能在相同或实质相关的事项上对抗前客户
- **保密信息**: 不能使用前客户的保密信息
- **重大相关**: 事项之间的重大相关性判断
- **同意豁免**: 获得客户同意的条件和程序

### AI工具使用的前客户保护
- **历史数据**: AI工具可能保留的历史客户数据
- **模式识别**: AI可能识别出的客户模式信息
- **交叉引用**: 不同客户案件的交叉引用风险
- **记忆效应**: AI工具的"记忆"和学习效应

### 合规检查要点
- [ ] 是否有前客户信息保护措施
- [ ] 是否避免了历史数据的不当使用
- [ ] 是否包含前客户利益保护提醒
- [ ] 是否有适当的信息隔离机制

## Rule 5.5 - 未经授权的法律执业 (Unauthorized Practice)
### 执业授权要求
- **执业资格**: 必须具备相关司法管辖区的执业资格
- **临时执业**: 临时性执业的条件和限制
- **多州执业**: 跨州执业的规则和要求
- **专业限制**: 专业领域的执业限制

### AI工具的执业授权问题
- **法律建议**: AI不能独立提供法律建议
- **客户代表**: AI不能代表客户进行法律行为
- **文件准备**: AI生成文件必须经律师审查
- **法庭出庭**: AI不能替代律师出庭

### 合规检查要点
- [ ] 是否明确AI仅为辅助工具
- [ ] 是否强调律师最终责任
- [ ] 是否避免了AI独立执业的暗示
- [ ] 是否包含执业资格要求说明

## Rule 1.4 - 与客户的沟通 (Communication with Client)
### 沟通义务
- **及时沟通**: 及时回应客户的合理询问
- **充分告知**: 告知客户案件的重大发展
- **决策参与**: 让客户参与重大决策
- **费用透明**: 明确说明费用结构和计算方法

### AI工具使用的沟通要求
- **工具解释**: 向客户解释AI工具的使用
- **局限性说明**: 说明AI工具的局限性
- **验证程序**: 说明律师验证AI输出的程序
- **风险告知**: 告知使用AI工具的风险

### 合规检查要点
- [ ] 是否包含客户沟通要求
- [ ] 是否说明了AI工具的作用和局限
- [ ] 是否明确了律师验证的重要性
- [ ] 是否包含风险披露要求

## Rule 1.15 - 客户财产的保管 (Safekeeping Property)
### 财产保管义务
- **信托责任**: 对客户财产的信托义务
- **分别保管**: 客户财产与律师财产的分别保管
- **完整记录**: 客户财产的完整记录
- **及时交付**: 及时向客户交付财产

### AI工具相关的财产保管
- **数字资产**: 客户的数字资产和信息
- **知识产权**: 客户的知识产权保护
- **数据所有权**: 明确数据的所有权归属
- **访问控制**: 对客户信息的访问控制

### 合规检查要点
- [ ] 是否明确客户信息的所有权
- [ ] 是否有适当的信息保护措施
- [ ] 是否明确信息使用的限制
- [ ] 是否包含信息归还的程序

## Rule 7.3 - 律师广告和招揽 (Advertising and Solicitation)
### 广告规范
- **真实性**: 广告内容必须真实准确
- **误导性**: 避免误导性的宣传
- **专业性**: 保持专业形象和语言
- **比较声明**: 避免不当的比较声明

### AI工具宣传的规范
- **能力声明**: 如实描述AI工具的能力
- **效果宣传**: 避免夸大AI工具的效果
- **专业地位**: 明确律师的主导地位
- **服务范围**: 准确描述AI辅助服务的范围

### 合规检查要点
- [ ] 是否避免了夸大宣传
- [ ] 是否明确了律师的主导作用
- [ ] 是否真实描述了AI工具能力
- [ ] 是否避免了误导性表述
